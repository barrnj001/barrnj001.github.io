Virtual Pet

A java project utilizing custom methods in order to simulate taking care of a pet. The pet has its own attributes that rise and fall according to a tick method.

Virtual Pet Shelter

A java project building off the virtual pet idea, but with multiple pets with varying attributes that must be attended to, all placed within a hashmap. Again, each need is controlled by indivdual methods and a tick method.

Virtual Pet Shelter: Amok

This java project takes the virtual pet shelter idea to another level: there are different types of pets, each with vary types of care. A certain breed of pet may not be able to benefit from the care that another does, so the implementation of interfaces and extensions is necessary.
